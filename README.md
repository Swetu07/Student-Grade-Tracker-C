# Student Grade Tracker in C

A console-based Student Grade Tracker developed in C using structures, functions, and file handling.

## Features

* Add Student
* Display All Students
* Search Student
* Update Student Marks
* Delete Student
* Calculate Percentage
* Assign Grade
* File Handling

## Technologies Used

* C Programming
* Structures
* Functions
* File Handling

## Project Structure

```
Student-Grade-Tracker-C/
├── main.c
├── student.c
├── student.h
├── README.md
└── .gitignore
```

## Compile

```bash
gcc main.c student.c -o Student-Grade-Tracker
```

## Run

### Windows

```bash
.\Student-Grade-Tracker.exe
```
